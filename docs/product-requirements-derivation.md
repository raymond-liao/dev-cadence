# 产品分析能力退役说明

Dev Cadence 曾包含从产品想法、User Journey、Feature、PRD 和 Business Architecture 到 Story Map 的产品分析与组合规划设计。该能力已经移出本项目。

本文件只保留历史链接的可导航性，不再定义当前产品行为。Dev Cadence 现在从实施工作项准入开始：

```text
新需求或已有卡片
-> backlog
-> 必要的单卡 work-item-analysis
-> feature-dev | bug-fix | refactor
```

目标仓库中已有的 PRD、User Journey、Business Architecture、Feature 或 Story Map 属于用户数据。安装或升级 Dev Cadence 不会读取、迁移或删除这些资产。
