# Work Item Planning Workflow 迁移说明

原 `work-item-planning` workflow 已退出当前能力模型：

- Story Map、Milestone、MVP、产品切片和产品组合规划已经移除；
- 单卡创建、已有卡准入、Backlog 登记、排序和生命周期职责迁移到 [`backlog`](backlog.md)；
- 单张 Story、Task 或 Bug 的详细定义继续由 [`work-item-analysis`](work-item-analysis.md) 负责。

本文件只保留历史链接，不是可执行 workflow。当前规则源为：

```text
src/workflows/backlog/SKILL.md
src/workflows/work-item-analysis/SKILL.md
```
