# Discovery Workflow 退役说明

Dev Cadence 不再提供 `discovery` workflow。产品探索、PRD、User Journey、Business Architecture、Feature 定义和其他产品分析职责已经移出本项目。

本文件只保留历史链接的可导航性，不是可执行流程，也不表示目标仓库必须删除已有产品文档。当前可安装 workflow 以 `src/workflows/**/SKILL.md` 为准。

需要进入 Dev Cadence 实施的工作必须形成符合卡片契约的 Story、Task 或 Bug，并通过 [`backlog`](backlog.md) 登记。
