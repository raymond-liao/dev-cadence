## Workflow

### Stage 1: Verify Citation Coverage

Check:

- Does every major conclusion cite a source ID or analysis artifact?
- Are proposal claims tied to `SRC-PROPOSAL-README`?
- Are external claims tied to real collected sources?

### Stage 2: Verify Argument Validity

Check:

- Does the cited evidence support the conclusion?
- Are recommendations overclaiming beyond available evidence?
- Are tradeoffs represented fairly?
- Are there contradictions across analysis files?

### Stage 3: Identify Report Constraints

Create instructions for the Reporter:

- Which findings can be used confidently.
- Which findings must be framed as assumptions.
- Which claims should be omitted or softened.
- Which gaps require human follow-up.

### Stage 4: Produce Verification Report

Write the output to:

```text
{PROJECT_ROOT}/workspace/03.verification/argument_verification_report.md
```

## Output Requirements

Use this structure:

```markdown

## Required Corrections for Reporter

## Residual Risks
```
