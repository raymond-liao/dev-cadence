# Dev Cadence

Dev Cadence 是一套面向 AI coding agents 的工程交付纪律。它通过清晰的职责边界、交付节奏和证据机制，让 Human 与 AI 先对齐任务理解，再让 AI agents 在受控边界内完成工程工作，最后把结果和证据交回给 Human 验收与风险接受。

它的目标是让 AI agents 成为真实软件工程中可信赖的交付参与者。
